# newt-calculator

An interactive calculator written in **Newt**.

This is an early example project showing Newt running a real script with:

- string output
- numeric input
- variables
- arithmetic
- print statements

## Code

```newt
print "Newt Calculator"

val a = input_number("a: ")
val b = input_number("b: ")

print a + b
print a - b
print a * b
print a / b
